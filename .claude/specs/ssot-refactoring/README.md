## 배경
채널 연동 후 대시보드 탭들의 데이터 매칭이 안 되고, SSOT 기준점이 없어 하드코딩이 30+ 파일에 산재.
채널 라벨, 컬러, 메트릭 라벨, 채널 그룹 등이 각 파일마다 중복 정의되어 있었음.

## 목표
- 모든 채널/메트릭/컬러 상수를 `src/constants/`로 중앙화
- 컴포넌트, API, 서비스에서 하드코딩 완전 제거
- 채널 추가/제거/변경 시 한 곳만 수정하면 전체 반영
