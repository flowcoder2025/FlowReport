# Decision: React Hooks 순서 수정

> 날짜: 2026-02-12
> 상태: 적용됨

## 문제
워크스페이스 진입 시 런타임 에러 발생:
- "Rendered more hooks than during the previous render"
- 위치: overview-view.tsx 라인 51

## 원인
Early return 이후에 useMemo hooks가 위치하여 렌더링 간 hooks 호출 수 불일치

## 해결 방안
모든 useMemo hooks를 early return 전으로 이동

## Before
```
hooks 4개 → early return → useMemo 4개
로딩 중: 4개 호출
로딩 후: 8개 호출 → 에러
```

## After
```
hooks 4개 → useMemo 4개 → early return
항상: 8개 호출 → 정상
```

## 영향 범위
- 파일: src/components/dashboard/views/internal/overview-view.tsx
- 로직 변경: 없음 (순서만 변경)
