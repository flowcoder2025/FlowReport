# Epic: 워크플레이스 메뉴 UX 개선

> 상태: 진행중 (83%)
> 시작일: 2026-02-11
> 마지막 업데이트: 2026-02-11

## 배경

Business Panel 분석 결과 (Drucker, Christensen, Meadows, Doumont):
- 현재 대시보드가 "데이터 표시 중심"으로 "의사결정 지원" 부족
- 메뉴 7개가 과다하며 일부 중복 (Performance/Content)
- 피드백 루프 시스템 부재
- 경쟁사 비교 기능 미구현

## 목표

1. 데이터 대시보드 → **의사결정 대시보드** 전환
2. 메뉴 단순화: 7개 → 6개
3. 피드백 루프 시스템 구축
4. 경쟁사 비교 기능 완성

## Phase 구성

| Phase | 이름 | 상태 | 설명 |
|-------|------|------|------|
| 1 | 긴급 개선 | ✅ 완료 | 헤드라인 요약, 권장 조치 |
| 2 | 중요 개선 | ✅ 완료 | 경쟁사 비교, 피드백 루프, KPI 리팩토링 |
| 3 | 구조 개선 | ✅ 완료 | Performance-Content 통합, 상관관계 차트 |
| 4 | 안정화 | 🔄 진행중 | API 연결, DB화, 문서화 |

## 주요 결정

1. **Performance-Content 통합**: 탭 구조로 통합 (메뉴 7→6개)
2. **상관관계 분석**: Pearson 상관계수 + 추세선 방식 채택
3. **경쟁사 비교**: 별도 DB 모델(Competitor) 생성
