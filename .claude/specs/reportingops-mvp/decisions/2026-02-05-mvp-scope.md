# Decision: MVP 범위 결정

> 일자: 2026-02-05
> 상태: 확정

## 컨텍스트

PRD에서 다양한 커넥터와 기능을 정의했으나, MVP 범위를 좁혀야 함.

## 결정 사항

### 1. MVP 커넥터 (3개)
- **GA4** - 트래픽/DAU/WAU (서비스 계정 JSON)
- **Meta (Instagram/Facebook)** - SNS 지표 (OAuth)
- **YouTube** - 영상 지표 (OAuth)

> SmartStore/Coupang은 Post-MVP로 연기

### 2. 인증 방식
- **Google OAuth만** MVP에서 지원
- 이메일/비밀번호, GitHub OAuth는 Post-MVP

### 3. 기능 범위
**포함:**
- 워크스페이스 기반 멀티테넌시
- RBAC (Admin/Editor/Viewer)
- CSV 업로드 (연동 불가 채널용)
- 주간/월간 대시보드
- PDF Export

**제외 (Post-MVP):**
- 실시간 알림
- 고급 권한 공유
- 자동 인사이트 생성

## 근거

- 빠른 MVP 출시 우선
- 가장 많이 사용되는 채널부터 지원
- fdp-backend-architect 아키텍처 활용으로 개발 속도 향상
